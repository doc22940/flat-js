# flat-js
Flat's javascript library

##Installation
 
```Shell
bower install katmore/flat-js --save
```

##Usage

```html
<script src="./components/flat-js/src/flat.js"></script>
```

##CDN

```html
<script src="https://cdn.rawgit.com/katmore/flat-js/master/src/js/flat.js"></script>
```
